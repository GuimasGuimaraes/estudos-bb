Git & DevOps

ParDiferença que a banca cobra
merge × rebasemerge preserva histórico com commit de mesclagem; rebase reescreve, deixando linear. Não rebasear branch pública.
fetch × pullfetch só baixa as referências; pull = fetch + merge no branch atual.
contêiner × VMcontêiner compartilha o kernel do host (leve); VM virtualiza hardware com SO completo (isolamento maior).
CI × CDCI = integrar + testar a cada commit; CD = entregar/implantar automaticamente o que passou.
Scrum × KanbanScrum: sprints fixas, papéis (PO prioriza backlog, SM facilita). Kanban: fluxo contínuo, limite de WIP.
